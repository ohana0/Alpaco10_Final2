# Alpaco10_Final_Project
알파코캠퍼스 2조

주제 : 뇌종양 분석과 RAG를 활용한 자동 의학 소견 생성 시스템 구축

목적: 
 - MRI 사진에서 3D segmentation 모델로 뇌종양(tumor) 발견
 - 전문서적 전체 pdf에서 관련 지식 정보 가져오기
 - 통합해서 Medical Letter 출력

Dataset:
 - BRATZ 데이터셋
 - PubMed conversation 데이터셋
 - Brain Section 데이터셋

Model:
 - UNet
 - SwinUnetR
 - Gemma2-2B

Tech Stack:
 - Python
 - pytorch
